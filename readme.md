# User Management System

## Backend Prerequisite

1. go to `/directus` and run this command

```
docker compose up -d
```

2. password can be find on `docker-compose.yml`


## Frontend Prerequisite
1. go to `/frontend-app`
```
cd /frontend-app
```
2. run this command
```
npm install
npm run dev
```
3. visit 
```
http://localhost:5173/
```
